## Performance is a design choice

Fast sites feel premium because they respect attention. Every animation, image, and script should have a reason to exist.

## Keep the server doing the heavy lifting

App Router server components reduce client weight and help separate data loading from interaction. That separation is especially useful in portfolio sites that mix storytelling with dynamic sections.

## Use motion with restraint

Motion should direct the eye, not compete for it. A few carefully timed transitions are enough to create a memorable experience.

## Measure the result

The real goal is not a clever stack. It is a site that opens instantly, reads clearly, and stays responsive on every device.
